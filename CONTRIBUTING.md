# Contributing to OLX Scraper

Thank you for considering contributing to OLX Scraper! 🎉

## Getting Started

1. **Fork** the repository and clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/olx-scraper.git
   cd olx-scraper
   ```

2. **Create a branch** for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Set up your development environment:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # or .venv\Scripts\activate on Windows
   pip install -r requirements.txt
   pip install flake8
   ```

## Making Changes

- Follow [PEP 8](https://peps.python.org/pep-0008/) style guidelines
- Keep lines under 120 characters
- Add comments for complex logic
- Test your changes manually before submitting

## Submitting a Pull Request

1. Push your branch: `git push origin feature/your-feature-name`
2. Open a PR against the `main` branch
3. Fill in the PR template completely
4. Wait for review — we will respond within a few days

## Reporting Bugs

Use the [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md) template when opening issues.
